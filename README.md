# Neovim configuration

This is my basic Nvim configuration. It uses [lazy-nvim](https://www.lazyvim.org/).


## Plugins

1. Theme - [Tokyonight](https://github.com/folke/tokyonight.nvim).
1. [Lualine](https://github.com/nvim-lualine/lualine.nvim) with [dev icons](https://github.com/nvim-tree/nvim-web-devicons).
1. [Telescope](https://github.com/nvim-telescope/telescope.nvim)
1. [Tree-Sitter](https://github.com/nvim-treesitter/nvim-treesitter)
1. [Which Key](https://github.com/folke/which-key.nvim)

Additionally, it uses LSP (Language Server Protocol).
